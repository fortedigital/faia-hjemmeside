# FAIA Landing Page Design

## Overview
Clean, minimal landing page for Forte AI Accelerator. React + Vite, SCSS modules. Colors: black, white, #511e29 only. Forte logo from fortedigital.com.

## Structure
Three-tab SPA: Hjem (hero + chat), Caser (customer cases), Hva vi gjor (what we do).

## Hjem
- Header: Forte logo left, three nav tabs right
- Hero: Large left-aligned "AI Accelerator" headline + elevator pitch subtitle. Generous whitespace
- Chat: Elegant chat box, white bg, subtle #511e29 border. Mocked responses from Notion content. Placeholder: "Fortell oss om din utfordring..."

## Caser
- Intro text + cards for example cases from the four tracks
- Each card: title, track label (A/B/C/D), short description
- Cases: Hendelsesrapportering (A), Anbudsanalyse (B), Tale-til-tekst (C), Legacy + AI-sok (D)

## Hva vi gjor
- Intro about 6-week model
- Four tracks as clean sections
- "After 6 weeks" - three outcomes (scale, continue, stop)
- Value proposition for decision makers

## Visual Language
- Font: Inter
- Spacing: Very generous, Scandinavian minimalism
- No decorative elements, icons only where functional
- Logo: Forte SVG + "AI Accelerator" subtitle
